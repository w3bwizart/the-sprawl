# Create PPTX Skill Directory

## Goal
The `Core/skills/create-pptx/` directory encapsulates the logic and resources required for programmatic PowerPoint generation.

## What to Place Here
- `SKILL.md`: Metadata.
- `logic.py`: The Python script.
- `templates/`: (Optional) Master templates.

## Examples of Tasks & Processes

1.  **Sales Manager**: "Take this meeting transcript and generate a summary deck using the `create-pptx` skill."
2.  **Finance Analyst**: "Create a quarterly review presentation with a slide for each region's P&L."
3.  **PMO Lead**: "Generate a kick-off deck for Project Alpha with a timeline slide."
4.  **Marketing**: "Update the 'Company Overview' deck by replacing the old stats with new projections."
5.  **Training Coordinator**: "Build a templated training module deck from this markdown manual."
6.  **Product Owner**: "Create a feature roadmap slide deck based on the upcoming epics."
7.  **Consultant**: "Generate a client proposal deck using the 'Standard_Minimal' template."
8.  **Event Planner**: "Make a looping slideshow for the lobby screen listing speakers."
9.  **Researcher**: "Visualize these JSON survey results as a series of chart slides."
10. **Executive**: "Create 5 slides from these 5 bullet points, utilizing high-quality background images."
