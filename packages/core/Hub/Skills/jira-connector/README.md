# Jira Connector Skill Directory

## Goal
The `Hub/Skills/jira-connector/` directory contains the specific logic for interaction with Atlassian Jira.

## What to Place Here
- `SKILL.md`: Metadata.
- `jira_client.py`: The logic handling API calls.

## Examples of Tasks & Processes

1.  **Project Manager**: "Create a new Task in project PROJ titled 'Update Readme'."
2.  **Developer**: "Transition ticket PROJ-42 to 'In Progress'."
3.  **QA Engineer**: "Log a bug on ticket PROJ-99 with the description 'Login fails'."
4.  **Scrum Master**: "Get the status of all high-priority tickets."
5.  **Product Owner**: "Update the description of Epic PROJ-1."
6.  **Support Agent**: "Link this conversation thread to Jira ticket SUPPORT-101."
7.  **Release Manager**: "Comment on ticket PROJ-55: 'Deployed to Staging'."
8.  **Team Lead**: "Assign all unassigned tickets to the 'Triage' user."
9.  **Auditor**: "Fetch the history of ticket PROJ-12."
10. **User**: "What is the status of my request PROJ-77?"
