# Salesforce Lookup Skill Directory

## Goal
The `Hub/Skills/salesforce-lookup/` directory enables the agent to query CRM data.

## What to Place Here
- `SKILL.md`: Metadata.
- `sf_fetch.py`: Logic to query SOQL or fetch records.

## Examples of Tasks & Processes

1.  **SDR**: "Who is the primary contact for 'Acme Corp'?"
2.  **Account Executive**: "What is the current stage of the 'Big Deal' opportunity?"
3.  **Sales Manager**: "List all opportunities closing this month."
4.  **Support**: "Check if 'XYZ Ltd' has an active support contract."
5.  **Marketing**: "Add this new contact to the 'Q1 Webinar' campaign."
6.  **Finance**: "Get the billing address for Account ID 001..."
7.  **VP Sales**: "Update the 'Next Steps' field on the 'MegaCorp' opportunity."
8.  **Lead Gen**: "Check if this email address already exists."
9.  **AM**: "Log a call activity against the 'ExampleCorp' account."
10. **Operations**: "Fetch the owner of the 'Global' territory."
