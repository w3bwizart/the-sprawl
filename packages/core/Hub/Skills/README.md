# Hub Skills Directory

## Goal
The `Hub/Skills/` directory houses **Expert Molecules**. These are specialized tools and integrations that give the agent capabilities specific to a domain.

## What to Place Here
- Skill folders (e.g., `jira-connector/`, `salesforce-lookup/`).
- Each folder must have a `SKILL.md` and the executable logic.

## Examples of Tasks & Processes

1.  **PMO**: Deploy `Hub/Skills/jira-connector/` to allow Project Managers to manage tickets.
2.  **Sales**: Use `Hub/Skills/salesforce-lookup/` to query CRM contacts.
3.  **Finance**: Add `Hub/Skills/xero-api/` to pull the latest invoice status.
4.  **Marketing**: Create `Hub/Skills/cms-poster/` to enable publishing blog posts directly.
5.  **HR**: Add `Hub/Skills/workday-lookup/` to check leave balances.
6.  **Engineering**: Create `Hub/Skills/aws-helper/` to check cloud resource status.
7.  **Support**: Add `Hub/Skills/twilio-sender/` to allow sending SMS updates.
8.  **Legal**: Create `Hub/Skills/docusign-check/` to see if a contract has been signed.
9.  **Operations**: Add `Hub/Skills/google-sheets-sync/` to push metrics into a shared sheet.
10. **Management**: Use `Hub/Skills/tableau-fetch/` to retrieve KPI dashboard snapshots.
