# Hub Scripts Directory

## Goal
The `Hub/Scripts/` directory holds **Raw Executables**. These are shell scripts, Python utility scripts, or other binaries.

## What to Place Here
- Shell scripts (`.sh`).
- Standalone Python scripts (`.py`).

## Examples of Tasks & Processes

1.  **DevOps**: Run `shared-utils.sh` to install common dependencies.
2.  **Data Engineer**: execute `etl-cleaner.py` to sanitize a CSV file.
3.  **SysAdmin**: Run `backup-logs.sh` to archive interaction logs.
4.  **Developer**: Use `generate-env.sh` to scaffold a local `.env` file.
5.  **Security**: Run `audit-scanner.sh` to check for hardcoded secrets.
6.  **PMO Admin**: Execute `archive-project.sh` to move old project folders.
7.  **Sales Ops**: Run `lead-import.sh` to bulk-load a CSV of leads.
8.  **Support**: Use `log-viewer.sh` to tail live session logs.
9.  **QA**: Run `test-runner.sh` to execute the unit tests.
10. **Maintenance**: Run `cache-clear.sh` to remove temporary files.
